# System Hardening Protocol - Proje Özeti

## Genel Bakış
Tek dosyalı HTML uygulaması olarak geliştirilmiş, "Monk Mode" yaşam tarzı yönetim paneli. Cyberpunk/taktiksel estetik ile kişisel gelişim takibi yapan bir sistem.

## Hedef Kullanıcı
- Kendini geliştirmek isteyen birey
- Disiplinli yaşam tarzı hedefleyen kişi
- Vücut geliştirme ve beslenme takibi yapan sporcu

## Ana Hedefler
1. **Kilo Hedefi**: 45kg → 60kg (Dirty Bulk protokolü)
2. **Günlük Kalori**: 3000 kcal hedef
3. **Antrenman**: Ev ortamında minimal ekipmanla güç antrenmanı
4. **Zihinsel Dayanıklılık**: 8 fazlı mental hardening protokolü

## Temel Özellikler
- Dashboard (Streak, İlerleme, Su/Uyku takibi)
- Antrenman sekmesi (Günlük program, set/tekrar girişi)
- Beslenme sekmesi (Kalori/makro takibi, öğün planı)
- Zihinsel Savaş (8 fazlı manifesto)
- Anatomi Lab (İnteraktif kas haritası)
- Gelişim (Kilo geçmişi, hacim grafikleri, ölçüler)

## Kapsam Dışı
- Bulut senkronizasyonu (localStorage kullanılıyor)
- Çoklu kullanıcı desteği
- Mobil uygulama (sadece responsive web)
